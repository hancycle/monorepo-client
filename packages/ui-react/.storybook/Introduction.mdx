# Hancycle Design System

Hancycle의 디자인 시스템에 오신 것을 환영합니다! 이 디자인 시스템은 Hancycle의 모든 제품에서 일관된 사용자 경험을 제공하기 위한 컴포넌트와 디자인 토큰을 포함하고 있습니다.

## 시작하기

디자인 시스템은 크게 두 가지 섹션으로 구성되어 있습니다:

### 1. 토큰 (Tokens)

디자인의 기본이 되는 요소들입니다:

- **Colors**: 브랜드 색상과 의미적 색상
- **Typography**: 텍스트 스타일과 크기
- **Effects**: 그림자, 투명도 등의 효과
- **Size**: 간격, 크기 등의 측정 단위
- **Icons**: 시스템에서 사용되는 아이콘

### 2. 컴포넌트 (Components)

재사용 가능한 UI 컴포넌트들입니다:

- **Typography**: 제목, 본문 등의 텍스트 컴포넌트
- **Loader**: 로딩 상태를 표시하는 컴포넌트
- 그 외 추가될 컴포넌트들...

## 사용 방법

```tsx
import { Title, Body, Loader } from "@hancycle/ui-react";

function MyComponent() {
  return (
    <div>
      <Title size="1">안녕하세요!</Title>
      <Body size="b-1">
        Hancycle 디자인 시스템을 사용하면 일관된 UI를 쉽게 구현할 수 있습니다.
      </Body>
      <Loader size="medium" color="primary" />
    </div>
  );
}
```

## 기여하기

디자인 시스템은 지속적으로 발전하고 있습니다. 새로운 컴포넌트나 개선사항이 있다면 팀과 논의해주세요.

## 문서 구조

왼쪽 사이드바에서 모든 컴포넌트와 토큰을 탐색할 수 있습니다:

- 각 컴포넌트는 Props 테이블과 사용 예시를 포함합니다
- 토큰은 카테고리별로 정리되어 있습니다
- Controls 탭에서 컴포넌트를 실시간으로 테스트해볼 수 있습니다
